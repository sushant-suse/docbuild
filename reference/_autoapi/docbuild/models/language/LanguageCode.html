docbuild.models.language.LanguageCode#

class docbuild.models.language.LanguageCode(/, **data: Any)#

Bases: pydantic.BaseModel

The language in the format language-country (all lowercase).

It accepts also an underscore as a separator instead of a dash. Use “*” to denote “ALL” languages

language: str = None#

The natural language in the format ll-cc, where ‘ll’ is the language and ‘cc’ the country.

model_config#

Configuration for the model, should be a dictionary conforming to Pydantic’s ConfigDict.

ALLOWED_LANGS: ClassVar[frozenset]#

Class variable containing all allowed languages.

__str__() → str#

Implement str(self).

__repr__() → str#

Implement repr(self).

__eq__(other: object | str | LanguageCode) → bool#

Implement self == other.

The comparison does NOT break the principle of equality: * Reflexive: a == b * Symmetric: a == b <=> b == a * Transitive: if a == b and b == c, then a == c

If you need to check for wildcar logic, use matches()

__lt__(other: object | str | LanguageCode) → bool#

Implement self < other.

Special properties: - “*” is always the “smallest” language - If self contains “*” and the other not, return True - If self and the other contains “*”, return False

__hash__() → int#

Implement hash(self).

For using ‘in sets’ or as dict keys

matches(other: LanguageCode | str) → bool#

Return True if this LanguageCode matches the other, considering wildcards.

The string ‘*’ matches any language:

>>> LanguageCode("*").matches("de-de")
True
>>> LanguageCode("de-de").matches("*")
True
