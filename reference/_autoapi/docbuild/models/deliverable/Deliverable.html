class docbuild.models.deliverable.Deliverable#

A class to represent a deliverable.

Usually called with a <deliverable> node from the XML configuration file. It contains information about the product, docset, language, branch, and other metadata related to the deliverable.

property productid: str#

Return the product ID.

property docsetid: str#

Return the docset ID.

property lang: str#

Returns the language and country code (e.g., ‘en-us’).

property language: str#

Returns only the language (e.g., ‘en’).

property product_docset: str#

Returns product and docset.

property pdlang: str#

Return product, docset, and language.

property pdlangdc: str#

Return product, docset, language and DC filename.

property full_id: str#

Return the full ID of the deliverable.

property lang_is_default: bool#

Check if the language is the default language.

property docsuite: str#

Return the product, docset, language and the DC filename.

property branch: str#

Return the branch where to find the deliverable.

Searches for the branch in the English language node first, then in the current language node.

Returns:

The branch name as a string.

Raises:

ValueError – If no branch is found

property subdir: str#

Return subdirectory or an empty string if not found.

property git: docbuild.models.repo.Repo#

Return the git repository.

Returns:

Raises:

property dcfile: str#

Return the DC filename or None if not found.

property basefile: str#

Return the DC filename without the DC prefix.

property format: dict[Literal['html', 'single-html', 'pdf', 'epub'], bool]#

Return the formats of the deliverable.

Normalize the format attributes to boolean values.

Returns:

A dictionary with format names as keys and boolean values.

Raises:

ValueError – If no format is found for the deliverable.
